CSS STYLE REFERENCE

Display and Positioninig in CSS

Property Description
Position Position property specifies the type of positioninig method used for an element
Display Display property specifies if/how an element is displayed

Position Attributes

Attributes Description
Static Default position value
Relative Positions element relative to its normal position
Absolute Positions element relative to its closet positioned parent element
Fixed Fixes an element to a specific position on a page (regardless of scrolling)
Sticky Keeps an element in normal document flow as user scrolls, but sticks to specified position as page is scrolled further

Display Attributes

Attributes Value
Inline Changes the display of Block elements to inline display
Block Changes the display of Inline elements to block display
Inline-Block Combine features of both Inline and Block
